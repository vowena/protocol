# Code of Conduct

## Our commitment

We want Vowena Protocol to be a respectful, welcoming project where contributors can work through hard technical problems together without hostility.

## Expected behavior

- Be thoughtful, direct, and respectful.
- Share enough context for others to review high-impact changes safely.
- Challenge ideas without attacking people.
- Help newer contributors understand protocol constraints instead of dismissing them.

## Unacceptable behavior

- Harassment, hate speech, or discrimination
- Personal attacks or intimidation
- Publishing private information without consent
- Repeated bad-faith disruption

## Scope

This applies in issues, pull requests, code review, discussions, chat, and any project space tied to Vowena Protocol.

## Reporting

If you need to report conduct concerns, contact the maintainers privately through GitHub. If that is not possible, email `security@vowena.xyz` with the subject prefix `[conduct]`.

## Enforcement

Maintainers may remove content, pause conversations, or restrict participation to keep the project safe and productive.
